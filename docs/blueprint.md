# **App Name**: Corporate Magnate

## Core Features:

- Loan Application Form: A multi-step form to collect user information for loan applications, ensuring all required fields are completed.
- Investment Plan Selector: Interactive tier cards that visually present different investment plans (Bronze, Silver, Platinum) with varying return rates and investment amounts, leading users to a specific investment form.
- Membership Signup Form: Collects information from new potential members; directs submission to corporatemagnate@outlook.com.
- Contact Form with Geolocation: A form to collect contact information and user inquiries; emails submission to corporatemagnate@outlook.com; also integrates an embedded map of Bayelsa.
- Floating 'Apply Now' Button: A call-to-action button that persists on the screen as the user scrolls, providing immediate access to the loan application form.
- Automated Email Confirmation: Automatic email sent upon submission of forms.

## Style Guidelines:

- Primary color: Deep blue (#003366) to convey trust and security.
- Background color: Very light blue (#E6F0FF), almost white, for a clean and professional backdrop.
- Accent color: Gold (#FFD700) for buttons and highlights, symbolizing prosperity and financial success.
- Body font: 'PT Sans', a humanist sans-serif, for a blend of modern and warm feel, suitable for continuous reading.
- Headline font: 'Space Grotesk', a modern sans-serif for impactful headlines and section titles.
- Clean and balanced layout with ample whitespace to focus user attention and ensure legibility.
- Subtle fade-in and slide-up animations to enhance user experience and guide visual flow.
